# Mapa IA por Areas

Mini juego web estatico (HTML/CSS/JS) listo para publicar en GitHub Pages.

## Publicacion en GitHub Pages

1. Asegura que el repositorio este en GitHub y la rama principal sea `main`.
2. Haz push de estos archivos (incluye `.github/workflows/pages.yml`).
3. En GitHub entra a `Settings` > `Pages`.
4. En `Source`, selecciona `GitHub Actions`.
5. Guarda y espera a que termine el workflow `Deploy static site to Pages`.

## URL publica

La URL final sera:

- `https://TU_USUARIO.github.io/TU_REPO/`

## Verificacion rapida

- La portada carga correctamente.
- Puedes seleccionar area/carrera/herramientas.
- El boton de PDF descarga el archivo sin errores.

## Desarrollo local

Abre `index.html` directamente en tu navegador.
